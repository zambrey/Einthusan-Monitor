Einthusan Monitor

Einthusan Monitor is a chrome extension to monitor www.einthusan.com for new HD movies.

View on GitHub Download .zip Download .tar.gz

Installation

  • To add this extension to your chrome browser, visit this page on Chrome Web Store.

Features

  • Lists the latest HD movies on www.einthusan.com.
  • Notifies as soon as new HD movies become available.
  • Allows to play movies straight from the extension.
  • Provides a way to search in all available HD movies.
  • Presents user with customization options for presentation as well as back end.
